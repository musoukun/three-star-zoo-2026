## 4. バースト判定テスト (5テスト)

| # | テスト内容 | 結果 |
|---|----------|------|
| 1 | うんちコマ7個以上でバースト → 星を1つ失う | PASS |
| 2 | うんちコマ6個以下はバーストしない | PASS |
| 3 | バースト時に星がない → 最高コスト動物を返却 | PASS |
| 4 | 同コスト動物が複数 → うんちコストが高い方を返却 | PASS |
| 5 | うんちコマちょうど7個でバースト | PASS |

**バーストルール検証**:
- 閾値: うんちコマ **7個以上** でバースト
- 星あり → 星-1
- 星なし → 最高コスト動物を返却（同コストならうんちコスト高い方を優先）
  - 例: アフリカゾウ(cost=6, poops=3) vs ミナミシロサイ(cost=6, poops=2) → アフリカゾウが返却される
- バースト後: コイン=0、うんちコマ=0 に全リセット

## 5. ゲーム終了条件テスト (4テスト)

| # | テスト内容 | 結果 |
|---|----------|------|
| 1 | 星3つ + うんち6以下 → 勝利 | PASS |
| 2 | 星3つ + うんちちょうど6 → 勝利 | PASS |
| 3 | 星3つ + うんち7以上 → バーストで勝利ではない | PASS |
| 4 | 星2つでは勝利しない | PASS |

**勝利条件**: 星 >= 3 AND うんちコマ < 7（自分のターン終了時）

**重要ケース**: 星3つ持っていてもうんち7以上ならバーストが先に発動し、星が2つに減って勝利条件未達成になる。
