# 三ツ星動物園 🦁

ボードゲーム「三ツ星動物園」のWeb版。リアルタイムマルチプレイヤー対応。

## 遊び方

1. ルームを作成（プライベートルーム対応）
2. 他のプレイヤーがルーム一覧から入室
3. ホストが「ゲーム開始」ボタンを押してスタート（2〜4人）

### ゲームの流れ

1. **セットアップ**: 初期動物（レッサーパンダ・ペンギン）を檻に配置
2. **メインフェーズ**: ターン制で以下を繰り返す
   - うんちコマを受け取る
   - サイコロを振る（1個 or 2個）→ 対応する檻の効果が発動
   - お買い物（動物1匹 + 星1つまで購入可能）
   - うんち掃除（1コインで2個除去）
3. **勝利条件**: 星3つを獲得し、うんちコマ6個以下でターンを終了

### バースト

ターン終了時にうんちコマが **7個以上** → バースト！
- 星を持っていれば星を1つ失う
- 星がなければ最高コストの動物を1匹返却（同コストならうんちコストが高い方）
- バースト後、全コインと全うんちコマを銀行に返却

### 檻のレイアウト

```
上段:  1    2    3    4    5    6
下段: [11&12]  10    9    8    7
```

11と12は結合された1つの大きな檻。ダイスで11または12が出ると同じ檻の効果が発動。

### 動物一覧

| 動物 | コスト | うんち | 色 | 効果 |
|------|--------|--------|-----|------|
| レッサーパンダ | 0 | 0 | 赤/緑/橙 | 自分のレッサーパンダ1匹につき+1コイン、ジャイアントパンダがいれば+2 |
| コンゴウインコ | 0 | 0 | 赤/緑/橙 | 全員+1コイン |
| ペンギン | 2 | 1 | 青 | 全員: ペンギン2匹以上なら+3、それ以外+1 |
| ライオン | 3 | 0 | 赤 | 全員: ライオン2匹以上なら3コイン奪取、それ以外1コイン |
| ジャイアントパンダ | 3 | 2 | 紫 | 全員: 自分のジャイアントパンダ1匹につき+2コイン |
| カリフォルニアアシカ | 3 | 0 | 青 | +3コイン & 対象から2コイン奪取 |
| アミメキリン | 4 | 1 | 緑 | 全員+3コイン、隣接檻にキリンがいれば+1 |
| チーター | 5 | 0 | 赤 | 全員+3コイン（endタイミング） |
| アフリカゾウ | 6 | 3 | 緑 | 全員+7コイン |
| ミナミシロサイ | 6 | 2 | 緑 | 対象から星1つ奪取 |
| バンドウイルカ | 6 | 0 | 青 | 選択: +7コイン or 対象から5コイン奪取 |

### 配置ルール

- 1つの檻に最大2匹
- 同じ檻に同じ動物は配置不可
- 檻に既に動物がいる場合、新しい動物は既存動物と色が1つ以上一致する必要がある
- 同じ動物の2匹目は、1匹目と隣接する檻にのみ配置可能

## 技術スタック

| | 技術 |
|---|------|
| サーバー | Colyseus 0.15 / TypeScript / Node.js |
| クライアント | React 19 / Vite 6 / colyseus.js |
| テスト | Vitest (33テスト) |
| デプロイ | Railway (サーバー) / Vercel (クライアント) |

## ローカル開発

```bash
# サーバー起動 (ポート2567)
cd colyseus-prototype/server
npm install
npm run dev

# クライアント起動 (ポート3000)
cd colyseus-prototype/client
npm install
npm run dev
```

ブラウザで http://localhost:3000 にアクセス。複数タブで開くとマルチプレイのテストができます。

## テスト

```bash
cd colyseus-prototype/server
npm test
```

詳細なテスト結果は [doc/test-results.md](colyseus-prototype/doc/test-results.md) を参照。

## デプロイ

### Railway（サーバー）

GitHubリポジトリを連携し、Root Directory を `colyseus-prototype/server` に設定。Dockerfileを自動検出してビルド。`PORT`環境変数はRailwayが自動注入。

### Vercel（クライアント）

```bash
cd colyseus-prototype/client
vercel --prod
```

環境変数 `VITE_SERVER_URL` に `wss://<Railwayドメイン>` を設定。
